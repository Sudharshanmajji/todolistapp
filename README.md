# Java To-Do List (Console Based)

A simple console-based To-Do List application written in Java.  
This project helps practice **ArrayList, Classes, Objects, and Menu-driven programs**.

---

## 🚀 Features
- Add a task
- View tasks
- Mark a task as completed
- Delete a task
- Exit the app
